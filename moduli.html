Moduli¶

Script per eseguire lo skim dei file di input e salvare i risultati in file di output I file di input sono specificati in file di testo, organizzati per ere I file di output sono salvati in un percorso specificato, con un nome che include l’era

step0_skim.read_files_idx(files_idx)¶

Funzione che legge i file di testo contenenti i nomi dei file di input e li organizza in un dizionario con le chiavi delle ere

Args:

files_idx (dict): Dizionario con le chiavi delle ere e i nomi dei file di testo

Returns:

dict: Dizionario con le chiavi delle ere e i nomi dei file di input

step0_skim.skim(files_dict, save_path='data/{eras}.root')¶

Funzione che esegue lo skim dei file di input e salva i risultati in file di output

Args:

files_dict (dict): Dizionario con le chiavi delle ere e i nomi dei file di input save_path (str): Percorso dove salvare i file di output, con un placeholder per l’era

Returns:

None
